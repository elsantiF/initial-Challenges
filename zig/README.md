# Zig Examples

To build all the examples, run:

```bash
zig build
```

To run a specific example, use:

```bash
zig run zig-out/bin/<example-name>
```

The example names are:

- `hello-world`
- `calculator`
- `number-guess`
- `todo`
- `palindrome`
- `fizzbuzz`

---

*Note:* These examples are made with Zig 0.16.0, may not work with other versions. Open an issue if you find any problem.
