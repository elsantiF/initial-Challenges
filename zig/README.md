# Zig Examples

To run an example, navigate to its folder and run `zig build run`:

```bash
cd <example_name>
zig build run
```

The example names are:

- `01_hello_world`
- `02_calculator`
- `03_number_guessing_game`
- `04_todo_list`
- `05_palindrome_checker`
- `06_fizzbuzz`

---

*Note:* These examples are made with Zig 0.16.0, may not work with other versions. Open an issue if you find any problem.
